Install Grapejuice using Burgerjuice

If you run into any problems, do not ask for help in the official Discord server or GitLab.

Burgerjuice is a installation script for Grapejuice to make installation as easy and idiot-proof as possible. It works on most mainstream Linux distributions. (Fedora, Arch and Ubuntu)

curl "https://raw.githubusercontent.com/maaa101/burgerjuice/main/burgerjuice.sh" | sh
